PacMen Exercise

This is a simple project that uses HTML and JavaScript to creates a PacMan image in a random starting postion when the create button is clicked. When the start button is clicked, the PacMen begin moving at random velocities and change directions once they reach the edge of the screen.

Dynamic Grid

This project creates a grid of 100 boxes in 10 row with 10 columns when the Create Grid button is clicked. When the Move button is clicked, the boxes toggle to orange then back to white one at a time, moving in order from 1 to 100.
